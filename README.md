## 🚀 Getting Started

To run this project locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/sabujghorai/HTML.git
   ```

2. Navigate into the folder:

   ```bash
   cd HTML
   ```

3. Open the HTML file:

   * Double click `index.html`
   * OR open with browser
