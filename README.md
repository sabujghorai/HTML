# HTML
# 🌐 HTML Learning Journey

## 📌 Project Overview
This repository documents my **Day 1 progress** in learning **HTML (HyperText Markup Language)**.  
HTML is the standard markup language used to create and structure web pages.

This project demonstrates the **basic structure of an HTML document** and the fundamental tags used to build a webpage.

---

## 🎯 Learning Objectives
On Day 1, I focused on understanding:

- What HTML is
- The structure of an HTML document
- Basic HTML tags
- How browsers render HTML pages

---

## 🏗️ HTML Document Structure

Every HTML page follows a standard structure.

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First HTML Page</title>
</head>
<body>

<h1>Welcome to My First Web Page</h1>
<p>This is my first step into web development.</p>

</body>
</html>
